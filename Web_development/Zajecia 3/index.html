HTML(HyperText Markup Language)
Standardowy język używany do tworzenia struktur i treści stron internetowych. Jest to język oparty na znacznikach, które definiują różne elementy na stronie, takie jak nagłówki, akapity, obrazy i linki.
HTML opisuje, jak treść powinna być wyświetlana w przeglądarkach internetowych, ale nie zajmuje się aspektami graficznymi. Przeglądarki interpretują kod HTML i wyświetlają zawartość strony w formie czytelnej dla użytkownika. Kod HTML składa się z otwierających i zamykających znaczników, które określają, jakie elementy znajdują się na stronie. Jest to fundamentalny element tworzenia stron internetowych i często używany w połączeniu z innymi technologiami, takimi jak CSS (Cascading Style Sheets) i JavaScript, aby dostosować wygląd i funkcjonalność witryny.

CSS (Cascading Style Sheets)
Język używany do określania wyglądu i formatowania treści na stronach internetowych. W skrócie, CSS definiuje, jak elementy HTML powinny być wyświetlane w przeglądarkach internetowych. Oto kilka kluczowych informacji na ten temat:

  1. CSS jest używany do kontroli stylu stron internetowych, takich jak kolor tła, kształt czcionek, odstępy między elementami i wiele innych aspektów wizualnych.
  2. CSS działa na zasadzie kaskadowania, co oznacza, że można zdefiniować różne style dla tych samych elementów HTML w różnych miejscach i CSS zostanie zastosowany w kolejności określonej w arkuszu stylów.
  3. Arkusze stylów CSS składają się z selektorów, które określają, do których elementów HTML mają być zastosowane style, oraz deklaracji, które określają konkretne właściwości stylu.
  4. CSS pozwala na tworzenie responsywnych projektów, które dostosowują się do różnych rozmiarów ekranów, co jest istotne w erze urządzeń mobilnych.
  5. Dzięki CSS można tworzyć animacje, efekty przejścia oraz wiele innych interaktywnych elementów, co przyczynia się do atrakcyjności stron internetowych.
  6. CSS jest nieodłącznym elementem tworzenia nowoczesnych stron internetowych i jest często używany w połączeniu z językiem HTML i JavaScript, aby osiągnąć pełnię możliwości projektowania i interakcji na stronie.

JavaScript
Wszechstronny język programowania używany głównie do tworzenia dynamicznych interakcji i funkcjonalności na stronach internetowych. Jest językiem skryptowym, który może być wykonywany w przeglądarkach internetowych, co pozwala na tworzenie interaktywnych elementów na stronach, takich jak formularze, animacje czy gry. JavaScript umożliwia dostęp do elementów i treści strony oraz ich modyfikację, co pozwala na dynamiczne zmiany bez konieczności przeładowywania strony. Język ten jest również stosowany poza przeglądarkami, na przykład w aplikacjach serwerowych, dzięki platformom takim jak Node.js. Działa na zasadzie zdarzeń, co oznacza, że reaguje na akcje użytkownika, takie jak kliknięcia myszą, wprowadzanie tekstu czy przewijanie strony. JavaScript jest oparty na obiektach, co ułatwia programistom tworzenie i zarządzanie obiektami oraz organizację kodu, co sprawia, że jest jednym z najważniejszych języków programowania w dziedzinie front-endowego i back-endowego rozwoju webowego.
